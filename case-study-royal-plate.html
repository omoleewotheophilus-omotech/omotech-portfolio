The Royal Plate

Luxury Steak & Grills Website — Multi-page, premium design, client-ready.

Project Overview

This project showcases a premium multi-page website built for The Royal Plate, a high-end steak and grills restaurant. The design is clean, bold, and modern, reflecting a luxurious dining experience while maintaining human warmth with imagery of the chef and dishes.

Role & Responsibilities

  • Lead Front-End Developer & Designer
  • HTML, CSS, JavaScript for multi-page layout
  • Responsive design for desktop & mobile
  • Implemented WhatsApp order integration & local cart persistence
  • Optional Google Sheets webhook for order tracking

Stack & Features

  • HTML / CSS / JS (vanilla)
  • Multi-page: Home, Menu, Order, Contact
  • Luxury black & gold theme
  • Mobile-first responsive layout
  • Portfolio-ready imagery
